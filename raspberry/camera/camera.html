Camera の使い方

video グループに加える

# sudo gpasswd -a user video

* failed to open vchiq instance というエラーが出なくなる

ラズベリーパイ用カメラモジュール

写真を取る

$ raspistill -o a01.jpg

python3 で写真を取る

# apt install python3-picamera

USB カメラ

USB で認識されているか調べる

	$ lsusb
	Bus 001 Device 006: ID 0411:0260 BUFFALO INC. (formerly MelCo., Inc.)
ソフトのインストール

# apt install fswebcam

写真を撮る

$ fswebcam photo.jpg
